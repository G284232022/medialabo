「Webコンテンツ作成」の成果

本テーマの学習目的

このテーマの学習目的は実際にWebページを作成し、その作成したものを自分なりに自由に装飾できるようにし、実際にユーザが入力した内容を反映させ、作成した数当てゲームのようなゲームも作成できるようになるという目的があります。

プロジェクト課題について

プロジェクト課題ではユーザに好きな料理のジャンルを入力してもらい、そのジャンルに関する情報をページ上に表示するWebページです。それぞれのジャンルで店舗名、住所、アクセス情報、キャッチコピーなどを表示するようにしました。また、ユーザが選択しやすいようにプルダウンメニューによってジャンルを選択しやすいようにし、簡易的に情報を表示するようにしました。ジャンルは合計で17種類設置しており、それぞれ情報を表示できるようになっています。

  • プロジェクト課題: グルメ情報

数当てゲーム

数当てゲームは、コンピューター側が決定した乱数をユーザーが数字を入力することによって合っているかどうかを判定するというプログラムです。ユーザの解答権は3回のみで、4回以上になってしまった場合その時点で終わりとなります。第4回ではJavaScriptファイルを編集し、予想メッセージである「○回目の予想:×」という文章をJavaScriptファイルにて表示するように編集しました。○の部分がきちんと予想回数と一致するように編集しました。第5回ではユーザの予想を固定している状態から入力させるようにイベント文にて変更し、一回一回予想が正解しているか判定するよう変更し、3回目以降は答えを表示し続けるように変更しました。また、3回以内に正解した場合は正解ですという文章を表示します。

演習の成果

  • 課題3-1: 数あてゲーム

毎回の実験で作成したページ

第1回 WWW と HTML

今回はHTMlにおける要素について学びました。大きく分けるとブロック要素とインライン要素の二つについて学びました。ブロック要素はインライン要素の中に収まることが不可能です。また、今回の課題ではもとより与えられたファイルの編集を行い、画像の掲載やハイパーリンクの実習を行いました。

演習の成果

  • 演習1-3: HTML文書の作成
  • 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの修飾

今回はCSSについて学びました。CSSのファイルを編集することで、Webページを装飾することができます。文章の色だけでなく、リンクの色も変えることができます。また、今回の演習ではWebブラウザの紹介をするファイルのCSSファイルを編集し、手本画像を参考に編集するということをやりました。

演習の成果

  • 練習問題2-3: Webブラウザの紹介

第3回 JavaScript の基本

今回はJavaScriptについて学びました。javaとJavaScriptでは変数宣言や比較、型変換など多くの異なる点があり、慣れないと難しい内容となっています。JavaScriptはデバックすることで間違っているか確認ができるのでデバッグが必須となっています。また、今回の課題では数当てゲームを作成しました。

演習の成果

  • 練習問題3-6: オブジェクトのメンバーを出力

第4回 DOM と JavaScript

今回はDOMプログラミングについて学びました。DOMプログラムでは実際のHTMLファイルに要素や文章を追加することができます。また、DOMによって実際に表示するページの文章も編集することができ、リンクも貼ることができます。今回の課題では、DOMによって写真を表示しさらに各地の緯度経度を表示するプログラムを作成しました。

演習の成果

  • 練習問題4-2~4-5: DOMプログラミング演習

第5回 イベント処理

今回はイベント文について学びました。イベント文を設置するとボタンやプルダウンメニューを作成できます。そのボタンを押すと背景色を変更したりすることができます。また、今回の演習ではイベントが発生する条件を変更し、さまざまな方法でランダムな色に変色するということをやりました。

演習の成果

  • 練習問題5-2: はじめてのイベント
  • 練習問題5-3: イベントの種類
