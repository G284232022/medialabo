本テーマの学習目的

プロジェクト課題について

数当てゲーム

数当てゲームは、コンピューター側が決定した乱数をユーザーが数字を入力することによって合っているかどうかを判定するというプログラムです。ユーザの解答権は3回のみで、4回以上になってしまった場合その時点で終わりとなります。第4回ではJavaScriptファイルを編集し、予想メッセージである「○回目の予想:×」という文章をJavaScriptファイルにて表示するように編集しました。○の部分がきちんと予想回数と一致するように編集しました。

演習の成果

  • 課題3-1: 数あてゲーム
  • 練習問題3-6: オブジェクトのメンバーを出力

毎回の実験で作成したページ

第1回 WWW と HTML

今回はHTMlにおける要素について学びました。大きく分けるとブロック要素とインライン要素の二つについて学びました。ブロック要素はインライン要素の中に収まることが不可能です。また、今回の課題ではもとより与えられたファイルの編集を行い、画像の掲載やハイパーリンクの実習を行いました。

演習の成果

  • 演習1-3: HTML文書の作成
  • 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの修飾

今回はCSSについて学びました。CSSのファイルを編集することで、Webページを装飾することができます。文章の色だけでなく、リンクの色も変えることができます。また、今回の演習ではWebブラウザの紹介をするファイルのCSSファイルを編集し、手本画像を参考に編集するということをやりました。

演習の成果

  • 練習問題2-3: Webブラウザの紹介

第3回 JavaScript の基本

今回はJavaScriptについて学びました。javaとJavaScriptでは変数宣言や比較、型変換など多くの異なる点があり、慣れないと難しい内容となっています。JavaScriptはデバックすることで間違っているか確認ができるのでデバッグが必須となっています。また、今回の課題では数当てゲームを作成しました。

演習の成果

  • 練習問題3-6: JavaScript演習

第4回 DOM と JavaScript

今回はDOMプログラミングについて学びました。DOMプログラムでは実際のHTMLファイルに要素や文章を追加することができます。また、DOMによって実際に表示するページの文章も編集することができ、リンクも貼ることができます。今回の課題では、DOMによって写真を表示しさらに各地の緯度経度を表示するプログラムを作成しました。

演習の成果

  • 練習問題4-2~4-5: DOMプログラミング演習
