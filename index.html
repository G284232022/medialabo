数当てゲーム

数当てゲームは、コンピューター側が決定した乱数をユーザーが数字を入力することによって合っているかどうかを判定するというプログラムです。ユーザの解答権は3回のみで、4回以上になってしまった場合その時点で終わりとなります。

第1回 WWW と HTML

今回はHTMlにおける要素について学びました。大きく分けるとブロック要素とインライン要素の二つについて学びました。ブロック要素はインライン要素の中に収まることが不可能です。また、今回の課題ではもとより与えられたファイルの編集を行い、画像の掲載やハイパーリンクの実習を行いました。

第2回 CSS による Web ページの修飾

今回はCSSについて学びました。CSSのファイルを編集することで、Webページを装飾することができます。文章の色だけでなく、リンクの色も変えることができます。また、今回の演習ではWebブラウザの紹介をするファイルのCSSファイルを編集し、手本画像を参考に編集するということをやりました。
